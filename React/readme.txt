React is a UI library hich helps us to write SPA ( single page apllication) which is efficient 
Recat application comprised pf multiple components all these componensts are organized in the tree struture format
In react bable compiler will convert JSX code into javaScript code 

-----------------------------------------------------------------------


To create react application we use a tool creat-react-application

1. To install create-react-app, 
open cmd and type below command 
> npm install create-react-app -g

2. To create react application 
> create-react-app counterdemo

3. To create application without installing create-react-app 
> npx create-react-app counterdemo

-----------------------------------------------------------------------

To run react application 
> cd counterdemo
> npm start
