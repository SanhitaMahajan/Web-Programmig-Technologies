# In javaScript there are 2 browser specific object to store properties 

1. Local Storage 
-- The properties will be availabe even after closing the browser window. It will remain int the object till you delete it explicitly 

2. Session Storage 
-- The properties will be availabe till the browser window is open. You may delete it by using clear or removeItems.

# Functions in local storage & session storage are - 
setItem
getItem
removeItems
clear
